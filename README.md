# Highly Available Web Application

## Project Overview
This project demonstrates the deployment of a highly available web application on AWS using an Application Load Balancer.

## AWS Services Used
- Amazon EC2 (2 Instances)
- Amazon VPC
- Public Subnets
- Internet Gateway
- Route Table
- Application Load Balancer

## Features
- Website hosted on two EC2 instances.
- Load Balancer distributes incoming traffic across both instances.
- High Availability and Fault Tolerance.
- Source code managed using Git and GitHub.
